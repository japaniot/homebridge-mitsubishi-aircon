# homebridge-mitsubishi-aircon

Homebridge plugin for Mitsubishi air conditioners.

## Usage

```js
"platforms": [
  {
    "platform": "MitsubishiAircon",
    "useDryForCool": false,
    "user": "USERNAME",
    "pass": "PASSWORD"
  }
]
```
